# Portfolio - Samia Regrai

A modern, bilingual portfolio website showcasing my journey as a Master's student in Business Intelligence & Big Data Analytics.

## 🌟 Features

- **Bilingual Support**: Seamless French/English language switching
- **Dark/Light Mode**: Toggle between elegant pastel pink and dark grey themes
- **Responsive Design**: Fully optimized for mobile, tablet, and desktop
- **GitHub Integration**: Automatically fetches and displays projects from GitHub API
- **Animated UI**: Particle effects, geometric shapes, and smooth transitions
- **Contact Form**: Integrated with Formspree for direct email submissions

## 🎨 Design

- **Light Mode**: Pastel pink aesthetic (#ff6b9d) with coral accents
- **Dark Mode**: Dark grey (#2b2b2b) with baby pink highlights (#ffb6c1)
- **Typography**: Poppins (body) and Playfair Display (headings)
- **Effects**: Glassmorphism cards, animated blobs, scroll progress bar

## 🛠️ Technologies

- HTML5, CSS3, JavaScript (Vanilla)
- Font Awesome Icons
- GitHub REST API
- Formspree (Contact Form)
- Google Fonts
